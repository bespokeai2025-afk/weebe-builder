BUILD IMAGE STUDIO WITHOUT DUPLICATING META ADS

IMPORTANT

Do not duplicate the Meta Ads tab.

Do not create a second Meta campaign manager.

Do not create duplicate ad campaign tables if equivalent campaign/ad tables already exist.

Use existing GrowthMind Campaign Factory and Meta Ads architecture where available.

OBJECTIVE

Build Image Studio as a creative asset engine only.

Image Studio should create and edit image assets.

Meta Ads should manage ad campaigns, audiences, budgets, publishing and performance.

Campaign Factory should connect both together.

CORRECT ARCHITECTURE

GrowthMind Strategy Centre
↓
Campaign Factory
↓
Creative Assets
   ├── Image Studio
   └── Video Studio
↓
Meta Ads / Google Ads / LinkedIn Ads
↓
Performance Analytics
↓
GrowthMind Optimisation

IMAGE STUDIO RESPONSIBILITY

Image Studio should handle:

Generate ad creatives
Generate social images
Generate product images
Generate blog images
Generate landing page hero images
Edit images
Create variations
Store image assets
Tag assets to campaigns

IMAGE STUDIO SHOULD NOT HANDLE:

Meta campaign launch
Ad account setup
Audience targeting
Budget setting
Ad set creation
Pixel/conversion setup
Meta performance reporting
Campaign optimisation

Those belong in Meta Ads / Ads Manager pages.

DATABASE RULE

Before creating any new table, audit existing tables for:

growthmind_campaigns
growthmind_campaign_drafts
growthmind_strategy_assets
growthmind_ad_creatives
growthmind_image_assets
growthmind_ads_accounts
growthmind_campaign_performance

If an equivalent table exists, extend it safely instead of duplicating it.

META ADS RELATIONSHIP

Image assets may be attached to Meta campaigns.

Add:

campaign_id
platform_hint = meta

But do not create Meta campaign logic inside Image Studio.

CAMPAIGN FACTORY INTEGRATION

Campaign Factory should be the bridge.

When creating a Meta campaign draft, allow:

Generate copy
Generate image creative
Generate video creative
Attach existing image
Attach existing video

The Meta Ads tab should then use those assets when publishing.

CONTENT STUDIO INTEGRATION

Content Studio can request:

Featured image
Social image
Ad image

but the generated image should be stored once in the image asset library and linked back to the content item.
