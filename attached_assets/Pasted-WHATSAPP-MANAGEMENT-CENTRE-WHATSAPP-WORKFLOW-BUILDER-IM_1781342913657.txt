WHATSAPP MANAGEMENT CENTRE + WHATSAPP WORKFLOW BUILDER

IMPORTANT

* Do not modify Retell functionality.
* Do not modify HyperStream voice functionality.
* Do not modify existing Builder functionality.
* Extend the existing Builder architecture.
* Reuse existing workflow engine wherever possible.
* Do not create a separate WhatsApp builder.
* Use the existing Builder shown in the platform.
* Generate implementation plan before coding.

OBJECTIVE

Transform the existing WhatsApp page into a complete WhatsApp Management Centre and allow the existing Builder to create WhatsApp conversation workflows by switching channel type.

The existing Builder must support:

Voice Agent Mode
WhatsApp Agent Mode

using the same workflow engine.
