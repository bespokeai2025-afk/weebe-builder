{ key, label }

* Write checkbox

* Checking write automatically checks read
* Unchecking read also unchecks write
* Write implies read

Create/edit form fields:

* Name
* Email
* Password
* Role: admin | user
* Permission grid

Only users with user_management write can access create/edit/delete/status controls.
