# SYSTEMMIND — FULL AGENT WORKFLOW EXECUTION ENGINE

You are upgrading the existing WEBEE SystemMind so that it can execute a complete agent setup from start to finish.

This is not a UI-only task.

SystemMind must become WEBEE’s internal workflow automation and integration engine, with capabilities comparable to n8n, but purpose-built for WEBEE voice agents, Retell AI, CRM integrations, lead management, call triggering, call queues, post-call processing and WEBEE page population.

The current SystemMind implementation has some existing agent setup capability. Audit and preserve everything that already works. Refine and extend the existing architecture rather than replacing working functionality.

## Non-negotiable outcome

A client must be able to select or reference an agent build inside the WEBEE Agent Builder, complete a guided setup process, and have SystemMind genuinely execute the setup.

At the end of the process, SystemMind must have:

1. Created or updated the agent in Retell AI.
2. Created the required dynamic variables.
3. Configured post-call data extraction fields.
4. Configured pre-call data retrieval.
5. Configured post-call data updates.
6. Connected the selected CRM.
7. Created and verified webhook endpoints.
8. Configured call triggers.
9. Configured call queues and calling rules.
10. Mapped call outcomes to WEBEE pages.
11. Tested the entire workflow.
12. Clearly shown the client which steps are completed, failed or still require information.

Do not mark a setup as complete merely because configuration records were saved. It is complete only when the external integrations and end-to-end workflow have been tested successfully.
