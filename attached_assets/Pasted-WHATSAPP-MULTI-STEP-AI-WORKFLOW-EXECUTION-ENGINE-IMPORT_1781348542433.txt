WHATSAPP MULTI-STEP AI WORKFLOW EXECUTION ENGINE

IMPORTANT

* Do not modify Retell functionality.
* Do not modify HyperStream voice functionality.
* Do not modify existing Builder functionality.
* Do not create a separate WhatsApp Builder.
* Reuse the existing Builder architecture.
* Preserve all existing Voice Agent functionality.
* Preserve all existing WhatsApp functionality.
* Generate implementation plan before coding.

OBJECTIVE

Upgrade WhatsApp agents from simple prompt-based responders into full multi-step AI workflow agents using the same Builder architecture currently used for Voice Agents.

The Builder must become the single source of truth for:

* Voice Agents
* WhatsApp Agents
* Future SMS Agents

====================================================

Most WhatsApp bots operate using:

Single Prompt
↓
User Message
↓
AI Response

* Unpredictable conversations

* Weak lead capture
* Difficult automation handling

====================================================

WHATSAPP EXECUTION MODES

Add:

Execution Mode

Options:

1. Structured Workflow
2. AI Assisted Workflow
3. Fully Autonomous AI

====================================================

MODE 1 – STRUCTURED WORKFLOW

Execute nodes sequentially.

Example:

Greeting
↓
Ask Name
↓
Save Variable
↓
Ask Budget
↓
Logic Split
↓
Book Appointment
↓
Confirmation

AI must remain inside the workflow.

Users cannot skip workflow stages.

====================================================

MODE 2 – AI ASSISTED WORKFLOW

Workflow controls conversation.

AI interprets user responses.

Example:

Node:
Ask Name

User may respond:

"My name is Nathan"
"Nathan Smith"
"Call me Nathan"

All responses must map correctly:

name = Nathan

Workflow then advances.

====================================================

MODE 3 – FULLY AUTONOMOUS AI

AI controls the conversation.

Workflow acts as:

* Guardrails
* Tools
* Knowledge
* Variables

Similar to OpenAI Assistant behaviour.
