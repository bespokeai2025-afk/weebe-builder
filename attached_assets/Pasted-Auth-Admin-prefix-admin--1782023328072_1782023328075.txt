Auth & Admin — prefix /admin

  ┌────────┬────────────────────────────┬──────────────┬────────────────────────────────────────────────────────────┐
  │ Method │            Path            │   Auth /     │                        Description                         │
  │        │                            │  Permission  │                                                            │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/login               │ Public       │ Log in with email + password. Returns accessToken,         │
  │        │                            │              │ refreshToken, and the user object (id, role, permissions). │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/logout/:id          │ Bearer       │ Log the admin/user out and invalidate their                │
  │        │                            │              │ session/refresh token.                                     │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/send-otp            │ Public       │ Send a one-time passcode to an admin (login/verification). │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/verify-otp          │ Public       │ Verify the submitted OTP.                                  │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/change-password/:id │ Bearer       │ Change the logged-in user's password (needs current        │
  │        │                            │              │ password).                                                 │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/forgot-password     │ Bearer       │ Start password reset: generate + email a reset token.      │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/verify-reset-token  │ Bearer       │ Validate a reset token before showing the new-password     │
  │        │                            │              │ form.                                                      │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/reset-password      │ Bearer       │ Set a new password using a valid reset token.              │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ POST   │ /admin/create-admin        │ Public       │ Register a new top-level admin account.                    │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ GET    │ /admin/get-all-admin       │ Bearer       │ List all admin accounts.                                   │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
  │ GET    │ /admin/get-admin/:id       │ Bearer       │ Get one admin by id.                                       │
  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤

  ├────────┼────────────────────────────┼──────────────┼────────────────────────────────────────────────────────────┤
