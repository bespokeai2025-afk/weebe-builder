FIX HIVEMIND PROACTIVE RECOMMENDATIONS

IMPORTANT

Do not rebuild HiveMind.

Do not rebuild GrowthMind.

Do not break existing HiveMind chat, briefing, tasks, recommendations, actions, GrowthMind, SystemMind, Builder, Smart Dash, Campaign Factory, Content Studio, Video Studio, WhatsApp or HexMail.

Fix the proactive recommendation behaviour only.

OBJECTIVE

HiveMind is not suggesting actions to the user.

HiveMind should proactively recommend what to do next, even when services are not connected yet.

Currently, it appears HiveMind may only respond when asked or may return nothing when data is missing.

Target behaviour:

HiveMind should always generate useful next-step recommendations based on the current workspace state.

====================================================

EXPECTED BEHAVIOUR

HiveMind should recommend actions in three situations:

1. No setup / empty workspace
2. Partial setup
3. Fully connected active workspace

====================================================

CASE 1 — EMPTY OR NEW WORKSPACE

If Business DNA is incomplete:

Recommend:

Complete Business DNA

Upload knowledge base

Choose onboarding path

Create first AI agent

Connect calendar if using Receptionist

Connect marketing platforms if using GrowthMind

Create first campaign draft

Show these as setup recommendations.

Do not return empty state.

====================================================

CASE 2 — PARTIAL SETUP

If Business DNA exists but integrations are missing:

Recommend:

Connect Google Analytics

Connect Search Console

Connect Meta Ads

Connect Google Ads

Connect WhatsApp

Connect Email

Connect Calendar

Connect CRM or choose WEBEE Smart Dash

Create first campaign

Create first receptionist agent

====================================================

CASE 3 — ACTIVE WORKSPACE

If data exists:

Recommend based on:

Calls

Bookings

Campaigns

WhatsApp

Provider health
