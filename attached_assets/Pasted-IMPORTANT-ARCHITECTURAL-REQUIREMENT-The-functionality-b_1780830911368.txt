IMPORTANT ARCHITECTURAL REQUIREMENT

The functionality below must ONLY apply to agents deployed using:

Go Live → Lead Generation

Do NOT apply this logic globally across the platform.

Do NOT modify:

* Receptionist agents
* Customer support agents
* Appointment booking agents
* Inbound agents
* FAQ agents
* Custom workflow agents

ACTIVATION CONDITION

When:

Agent Type = Lead Generation

Enable:

* CSV Variable Mapping
* Campaign Management
* Lead Intelligence
* Sentiment Analysis
* Lead Scoring
* Post-Call Lead Updates
* Campaign Analytics

When:

Agent Type ≠ Lead Generation

None of the above systems should load.

VARIABLE MAPPING

Only Lead Generation agents should receive:

Lead Context
↓
CSV Record
↓
Variable Mapping Engine
↓
Agent Variables
↓
Prompt Generation
↓
Retell

Example:

Agent Script contains:

{{first_name}}
{{company}}
{{industry}}

Before call starts:

CSV row:

{
"first_name":"Sarah",
"company":"Acme Ltd",
"industry":"Manufacturing"
}

Variables resolved:

{{first_name}} = Sarah

{{company}} = Acme Ltd

{{industry}} = Manufacturing

These values are injected only for Lead Generation agents.

Call Ends

↓

LEADS SECTION INTEGRATION

Only Lead Generation agents should write data into:

Leads Section
