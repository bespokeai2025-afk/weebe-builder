## AUDIT PRIORITY ORDER

To determine which endpoint should supply leads into WEBEE calling campaigns, inspect in this order:

1. GET /crm-data/get-crm-data
2. GET /call-output-data/get-userCall-lead
3. GET /call-output-data/get-all-calldata
4. GET /call-output-data/callbacks/pending
5. GET /call-output-data/all

For each endpoint, inspect:

* Total records returned
* Field names
* Status fields
* Lead identifiers
* Phone numbers
* Pagination support
* Record counts
* CRM classification fields

Then map the endpoint to:

* New Leads
* Tried To Contact
* Disqualified Leads
* Qualified Leads
* Callbacks
* Calls

inside WEBEE Smart Dash.
