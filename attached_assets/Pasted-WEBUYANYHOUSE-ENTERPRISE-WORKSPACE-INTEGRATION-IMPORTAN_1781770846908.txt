## PHASE 4 - LEADS MODULE

Do NOT create new pages.

Use existing WEBEE Leads functionality.

Create lead categories:

1. New Leads
2. Tried To Contact
3. Disqualified Leads
4. Qualified Leads
5. Callbacks

The imported Dynamics data must automatically map into these buckets.

Classification Rules:

If status contains:

* disqualified
* rejected
* unsuitable

Then:

lead_bucket = Disqualified

If status contains:

* tried to contact
* attempted contact
* no answer
* unable to contact

Then:

lead_bucket = Tried To Contact

If status contains:

* new
* fresh
* unworked

Then:

lead_bucket = New Leads

If qualification status becomes:

* positive
* neutral
* qualified

Then:

lead_bucket = Qualified Leads

If callback requested:

lead_bucket = Callbacks
