Do NOT blindly execute the root-level SQL files.

BACKGROUND

The SQL audit has identified SQL files located outside the standard supabase/migrations directory.

These files were not included in the previous migration reconciliation.

Some of the tables defined within them are actively referenced by the current codebase but do not exist in the live database.

These files require investigation before any SQL is executed.

---

PHASE 1 — RECONCILIATION

Review the following root-level SQL files:

* GROWTHMIND_CMO_MIGRATION.sql
* GROWTHMIND_SEO_EMAIL_MIGRATION.sql
* Any other root-level .sql files not already reconciled

For every object inside those files determine whether it is:

* Already implemented
* Missing
* Superseded
* Duplicated
* Obsolete

Do not assume every CREATE TABLE should be executed.

---

PHASE 2 — CODE VALIDATION

For every missing table determine:

* Is it actively referenced by the current code?
* Which files reference it?
* Which modules depend on it?
* Is it currently causing runtime failures?
* Is it silently failing because of missing tables?
* Is the feature currently disabled?
* Is there already an alternative implementation?

---

PHASE 3 — SAFE APPLICATION

Only if a table is genuinely required:

Apply only the missing schema objects.

Never recreate objects that already exist.

Never duplicate indexes.

Never duplicate RLS policies.

Never duplicate triggers.

Never duplicate functions.

All SQL must remain idempotent.

---

PHASE 4 — GROWTHMIND VALIDATION

After reconciliation verify the following GrowthMind modules:

* CMO Analysis
* Opportunity Scoring

Confirm:

* database tables exist
* queries succeed

---

PHASE 5 — POST VALIDATION

Verify:

* No duplicate schema objects
* No duplicate indexes
* No duplicate policies
* No duplicate triggers
* No duplicate functions
* No broken references
* No SQL errors
* No runtime errors
* No relation does not exist errors

RETURN

1. Root-level SQL files discovered
