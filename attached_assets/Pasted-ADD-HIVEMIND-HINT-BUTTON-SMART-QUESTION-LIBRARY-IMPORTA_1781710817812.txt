ADD HIVEMIND HINT BUTTON + SMART QUESTION LIBRARY

IMPORTANT

Do not rebuild HiveMind.

Do not break HiveMind chat, HiveMind voice, GrowthMind, SystemMind, AccountsMind, Builder, Smart Dash, Campaign Factory, WhatsApp, HexMail or existing recommendation routing.

This is a UI and intent-routing enhancement only.

OBJECTIVE

Add a Hint / Ask HiveMind button that gives users suggested phrases they can click or say to get useful results from HiveMind.

The hints should cover all major areas of the business and route to the correct intelligence engine.

CLICK BEHAVIOUR

When a user clicks a suggested question:

1. Fill the HiveMind input with that question.
2. Optionally auto-submit if user confirms.
3. Route it through the same HiveMind intent engine as typed/voice questions.
4. Create recommendation cards/tasks/actions where relevant.

====================================================

VOICE BEHAVIOUR

The same phrases should work by voice.

If user says one of the suggested phrases or similar wording, HiveMind should route correctly.

EMPTY WORKSPACE BEHAVIOUR

If the selected question requires missing data, HiveMind should not fail.

Example:

User clicks:
"Which campaigns are working?"

If no campaigns exist:

HiveMind should reply:

"You do not have active campaigns yet. I recommend creating your first campaign or connecting Meta/Google Ads."

Then create setup recommendations.
